Python Unit 1-Lab Runner

Description:

In Unit 1 we learned multiple things on how to make a game. We learned the basics of how to start the game and what the keys were to play the game. In Lab Runner in Unit 1 we had to make a blank white screen and then dsplay a character and the character had to be able to move around using the arrow keys or the WASD keys.

Concepts Learned:

Some of the concepts we learned was how to display a character on the wall and make the character able to move around the screen in all different directions. Another thing that we started to learn is collision and make it so when the character hits the side of the walls it stops instead of moving past off the screen.
